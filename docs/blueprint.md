# **App Name**: FraudSpotter

## Core Features:

- Data Input: Accept JSON data and an image via input fields.
- API Call: Send the data to a backend API for fraud analysis.
- Result Display: Display the risk assessment results, including risk level, score, flags, and AI reasoning.
- Audit Trail: Log all transaction details including ip address, Geo-location
- Audit Trail Display: Visually present the audit trail associated with the transaction.

## Style Guidelines:

- Primary color: Dark slate blue (#375EAB) to convey trust and seriousness.
- Background color: Light gray (#F0F4F8) to ensure content clarity and readability.
- Accent color: Soft coral (#F28482) to highlight risks and call attention to important information.
- Font pairing: Use 'Space Grotesk' (sans-serif) for headlines and 'Inter' (sans-serif) for body text.
- Employ clear and universally understood icons for risk levels and data categories.
- Design a clean and intuitive layout to guide users through the analysis process.
- Incorporate subtle animations to enhance user interaction and highlight critical results.
